# Gym Tracker - Project Scope

The Gym Tracker application is a full-stack Next.js web application designed to help users track their strength training progress. 

### Core Features:
- **Workout Logging**: Track exercises, sets, weight, and reps.
- **Progress Dashboard**: Visualize total volume, personal records (PRs), and weekly consistency using Recharts.
- **Workout History**: Chronological view of past workouts with detailed breakdowns.
- **Suggestion System (AI Coach)**: Algorithmic weight/rep suggestions based on previous performance (Progressive Overload logic).
- **Responsive Design**: Mobile-first, dark-themed UI built with Tailwind CSS.

### Tech Stack:
- Next.js 14 (App Router)
- MongoDB + Mongoose
- Tailwind CSS
- Recharts (for dashboard)
- TypeScript
